# rikettsie/scoop-bucket

Scoop bucket for tools by [rikettsie](https://github.com/rikettsie).

## Install

    scoop bucket add rikettsie https://github.com/rikettsie/scoop-bucket
    scoop install <formula>

## Available packages

| Package | Description |
|---------|-------------|
| `rdrop` | P2P streamed file transfer with ring-based access control |

## Update

    scoop update
    scoop update rdrop
